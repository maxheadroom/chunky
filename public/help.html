You can upload any big file via this application and provide a secret link to this file. Simply click on "show all files/upload" and pick an existing file or upload a new file.

After choosing a file to create a download link for fill out the form with appropriate information about this particular download. You can even choose to send your client an automated notification eMail about the new download link.

Optionally you can set a maximum number of downloads for this file. The link will become unavailable after the expire date or when the max download count has been reached.

The mail to your client will look like this:

show all available links | show expired links | show all files/upload
